Toggle sidebar plugin

What is a sidebar?

A sidebar for this plugin is considetared an area located in one side of the window or screen that contains categorized (or not) information. The toggle sidebar plugin "can help" to show and hide that information in the screen and response o clicks.

This page is intented to show how the plugin works and some requirements to get the job done.

How to use

1. Add the toggleSidebar.min.js file to project

2. Initialize the pluging at least after all HTML code is loaded or before close the body tag. The example shows the minimum object properties required to get a workable sidebar. 

window.addEventListener("DOMContentLoaded", (event) => {
	const sidebar = new ToggleSidebar({
		activationClass: "main-menu",
		containerNode: document.getElementById("menu"),
		openSidebarNode: document.getElementById("#icon-main-menu"),
	});
})

3. Add the required CSS code to make things happen

"How to use" sidebar demo

Remarks:

  • This sidebar uses a CSS code explained later, you need to do some experiments to make it work as you want.
  • As you will (or have) noticed, if you click on a link inside a sidebar, the sidebar will close. If the link leads to another page, no one cares, but if it links to any area of the same page, the sidebar can be awkward to read the content behind on narrow screens.
  • A sidebar, can appears from top or bottom (CSS control that)
  • You can set a sidebar open element into any sidebar. Try to open the Right sidebar
  • If you want to hide this sidebar, click outside of it, or open another sidebar, like the Left Sidebar
