Toggle sidebar plugin

What is a sidebar?

A sidebar for this plugin is considetared an area located in one side of the window or screen that contains categorized (or not) information. The toggle sidebar plugin "can help" to show and hide that information in the screen and response to clicks.

This page is intented to show how the plugin works and some requirements to get the job done.

How to use

1. Add the toggleSidebar.min.js file to project

2. Initialize the pluging at least after all HTML code is loaded or before close the body tag. The example shows the minimum object properties required to get a workable sidebar.

3. Add the requeired CSS code to makit things happen

Click here to show a working demo sidebar. There you will find all HTML and JS code used (checkout the JS object used) to get it up and running.

"How to use" sidebar demo

Code used:

toggleSidebar.addSidebar({
	name: "how-to-use-demo",
	activationClass: "show-how-to-use-demo",
	autoClose: true,
	containerNode: document.getElementById("how-to-use-plugin"),
	openSidebarNode: document.querySelector("#how-to-use-demo"),
});
					
.sidebar-container {
	background-color: darkgrey;
	border: 1px solid #000;
	height: 100dvh;
	overflow-y: scroll;
	padding: 1em;
	position: fixed;
	top: 0;
	transition-duration: 0.3s;
	transition-property: transform;
	transition-timing-function: cubic-bezier(0.22, 0.61, 0.36, 1);
	width: min(var(--sidebar-with), 90%);
	z-index: 1
}

#how-to-use-plugin {
	width: min(43em, 90%);
	left: 0;
	transform: translate(min(calc(43em * -1), 100%), 0);
}
/* Here is where the "activationClass"
property on JavaScript code makes
things happens */
body:is(.show-how-to-use-demo) #how-to-use-plugin {
	transform: translate(0, 0);
}

Remarks:

  • This sidebar uses a CSS code explained later, you need to do some experiments to make it work as you want.
  • As you will (or have) noticed, if you click on a link inside a sidebar, the sidebar will close. If the link leads to another page, no one cares, but if it links to any area of the same page, the sidebar can be awkward to read the content behind.
  • A sidebar, can appears from top or bottom (CSS control that)
  • You can set a sidebar open element into any sidebar. Try to open the Right sidebar
  • If you want to hide this sidebar, click outside of it, or open another sidebar, like the Left Sidebar
