Toggle sidebar plugin

What is a sidebar?

A sidebar for this plugin is considetared an area located in one side of the window or screen that contains categorized (or not) information. The toggle sidebar plugin "can help" to show and hide that information in the screen and response of clicks.

This page is intented to show how the plugin works and some requirements to get the job done.

How to use

1. Add the toggleSidebar.min.js file to project

2. Write the HTML code for the sidebar open node and the sidebar container node.

3. Initialize the pluging at least after all HTML code is loaded or before close the body tag. The example shows the minimum object properties required to get a workable sidebar. 

window.addEventListener("DOMContentLoaded", (event) => {
	const sidebar = new ToggleSidebar({
		activationClass: "show-main-menu",
		containerNode: document.getElementById("menu"),
		openSidebarNode: document.getElementById("#icon-main-menu"),
	});
})

4. Add the CSS code you need to make things happen. Here you will see the minimum necessary code (recomended) to make a working toggle sidebar. You can get a more convinient (dazzling) CSS code in this section.

.sidebar-container{
	/* Set CSS rules you want/need matching your design and next properties*/
	height: 100dvh; /* make the sidebar as tall as possible */
	overflow-y: auto; /* allow vertical scroll if needed*/
	position: fixed; /* sidebar will be positioned where you set*/
	top: 0; /* set sidebar on top of the screen*/
	width: max(15em, calc(100dvw - 4em)); /* standard sidebars size */
	z-index: 99 /* make sidebar view over other 98 layers */
}

#icon-main-menu{
	/* Set CSS rules you want/need matching your design*/
}

#main-menu{
	/* Set CSS rules you want/need matching your design and next properties*/
	display:none; /* hide sidebar */
}

body:is(.show-main-menu) #main-menu {
	display:block; /* show sidebar */
}

"How to use" sidebar demo

Remarks:

  • This sidebar uses a CSS code explained later, you need to do some experiments to make it work as you want.
  • A sidebar, can be set to the top, bottom or middle of the page (CSS controls that).
  • You can set a sidebar open element into any sidebar. Try it by opening the Right sidebar right now.
  • If you want to hide this sidebar, click outside of it, click on any open node for this sidebar, or open another sidebar, like the Left Sidebar
  • As you will (or have) noticed, if you click on a link inside a sidebar, the sidebar will close. If the link leads to another page, no one cares about hide sidebar status, but if it links to any area of the same page, the sidebar can be awkward to read the content behind on narrow screens.
  • When a sidebar is opened and clicking its opening node, the sidebar will close even if the node was not the same one used to open it.
  • Any sidebar trigger element can be any HTML node. A sidebar, open, close, or omit close node can be written in any tag, just try using semantic HTML.
  • Be careful with names in classes or id. Use a convention like "{action}-sidebar-{sidebarName}" to avoid conflicts with other elements, this is not handled by the plugin. This recommendation was not followed when writing the code for this page, sorry.ðŸ˜‰

const toggleSidebar = new ToggleSidebar();
toggleSidebar.addSidebar({
	name: "how-to-use-demo",
	activationClass: "show-how-to-use-demo",
	containerNode: document.getElementById("how-to-use-plugin"),
	openSidebarNode: document.querySelectorAll("span.how-to-use-demo-open-node")
});
/*Only relative working/transition properties are showing*/

:root {
	--sidebar-standard-with: 18em;
}

.sidebar-container {
	height: 100dvh;
	overflow-y: auto;
	position: fixed;
	top: 0;
	transition-duration: 0.3s;
	transition-timing-function: cubic-bezier(0.22, 0.61, 0.36, 1);
	width: min(var(--sidebar-standard-with), 90%); /* ** */
	z-index: 1
	/* **Property overridden later */
}

#how-to-use-plugin {
	transition-property: left;
	width: min(55em, calc(100vw - 4em));
	left: min(-55em, calc(-100vw - 4em));
}

on JavaScript object makes things happens */
